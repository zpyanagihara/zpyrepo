# Personal Website

This repository contains the source code for my personal website:

https://zpyanagihara.com

The site serves as a professional profile highlighting my experience across diagnostics, laboratory systems, medical devices, and biotech program leadership.

## Hosting

The website is a static site hosted using **GitHub Pages** and connected to a custom domain managed through **Namecheap**.

## Repository Contents

- `index.html` — main webpage structure
- `styles.css` — visual styling for the site
- `headshot.jpg` — profile image used on the site
- `Zachary_Yanagihara_Resume.pdf` — downloadable resume
- `CNAME` — configuration file linking the repository to the custom domain

Updates to the repository are automatically deployed to the live site through GitHub Pages.
